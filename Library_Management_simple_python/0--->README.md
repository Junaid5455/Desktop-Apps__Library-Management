# Library Management Software

A Python-based **Library Management System** developed during my internship using core Python programming concepts before Object-Oriented Programming (OOP). This project demonstrates practical implementation of **variables, functions, lists, dictionaries, loops, and conditional statements** to manage library operations efficiently.

## Features

* Book Management

  * Add new books
  * View all books
  * Update book quantity
  * Remove books

* Member Management

  * Register new members
  * View all members
  * View member details
  * Deactivate members

* Borrowing & Returns

  * Borrow books
  * Return books
  * View All Active Borrowings
  * Check Overdue Books

* Reports & Statistics

  * Library summary
  * Popular books
  * Active members
  * Genre distribution
  * Monthly borrowing reports

* Search System

  * Search books by title
  * Search books by author
  * Search books by genre
  * Search members by name

## Technologies Used

* Python (Basic Concepts)
* Functions
* Lists
* Dictionaries
* Loops
* Conditional Statements
* Date & Time Module

## Purpose

This project was created to strengthen programming fundamentals and demonstrate how real-world management systems can be built using basic Python concepts without OOP.

## Learning Outcomes

Through this project, I practiced:

* Data handling using lists and dictionaries
* Functional programming structure
* Record management
* User input handling
* Report generation
* Problem-solving with Python basics

## Note

This project is built entirely with procedural programming techniques and serves as a strong foundation before transitioning into Object-Oriented Programming concepts.
