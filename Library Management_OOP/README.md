# Library Management System (OOP - Python)

A console-based **Library Management System** developed in Python using **Object-Oriented Programming (OOP)** principles. This project is designed to manage books, members, borrowing, returns, reports, and search operations efficiently while demonstrating practical implementation of **classes, objects, inheritance-like structure, methods, and modular programming**.

## Features

### Book Management

* Add new books
* View all books
* Update book quantity
* Remove books

### Member Management

* Register new members
* View all members
* View member details
* Deactivate members

### Borrowing & Returns

* Borrow books
* Return books
* Due date tracking
* Fine calculation for overdue books
* Active borrowing records

### Reports & Statistics

* Complete library summary
* Most popular books
* Most active members
* Genre-wise book distribution
* Monthly borrowing reports

### Search System

* Search books by title
* Search books by author
* Search books by genre
* Search members by name
